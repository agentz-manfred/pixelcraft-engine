🎬 PixelCraft Engine — Animation

Frame-based animation, requestAnimationFrame, per-frame timing, GIF export.

1. Basic Animation (palette shift)

const anim = new Animation({
  frames: [frame1, frame2, frame3, frame4],
  fps: 3,
});
anim.play(canvas, { scale: 8 });

2. Rotation Animation

const frames = [sprite, sprite.clone().rotate90(), sprite.clone().rotate180(), sprite.clone().rotate270()];
const anim = new Animation({ frames, fps: 4 });

3. Per-Frame Duration

const anim = new Animation({
  frames: [
    { sprite: frame1, duration: 200 },   // fast
    { sprite: frame2, duration: 1000 },  // slow hold
    { sprite: frame3, duration: 200 },   // fast
  ],
});

4. Slow Animation (FPS = 0.5)

const anim = new Animation({ frames, fps: 0.5 }); // 2 seconds per frame

6. Animated GIF Export

const gifDataURL = animationToGIFDataURL(anim, { scale: 8 });

7. SpriteSheet ↔ Animation

// Export animation frames as spritesheet
const sheet = SpriteSheet.fromAnimation(anim);
// Import spritesheet back as animation
const restoredAnim = sheet.toAnimation({ fps: 4 });
